It emphasizes the need for a CRE account and the CLI installation as prerequisites. Additionally, it notes that for CI/CD environments, API key authentication should be used instead of browser login.. Key details: Prerequisites: CRE CLI installed, CRE account created, access to authenticator app; Login command: 'cre login'; Browser-based authentication process; Two-factor authentication options: authenticator app or biometric; Confirmation message upon successful login
- [https://docs.chain.link/cre/account/creating-account](https://docs.chain.link/cre/account/creating-account) — This page provides a step-by-step guide for creating an account on the Chainlink CRE platform, which is necessary for using the CLI and running commands. It covers two account creation options: creating a new organization or joining an existing one. Key steps include entering personal information, verifying email, setting a secure password, enabling two-factor authentication (2FA), and saving a recovery code. Important security practices and prerequisites are also highlighted.. Key details: Valid email address required; 6-digit verification code for email verification; Password must meet security requirements; 2FA setup options: biometric or authenticator app; Recovery code must be stored securely
- [https://docs.chain.link/cre/account/managing-auth](https://docs.chain.link/cre/account/managing-auth) — This page provides a comprehensive guide on managing CLI authentication for Chainlink's CRE platform. It covers browser-based login, API key authentication (including environment variable setup), session expiration handling, checking authentication status, and logging out. Key commands include 'cre login', 'cre whoami', and 'cre logout'. Important caveats include the requirement for Early Access approval for API key authentication and the need to re-authenticate after session expiration.. Key details: cre login command for browser-based login; API key set via CRE_API_KEY environment variable; cre whoami command to check authentication status; cre logout command to end session; Session expires requiring re-login
