- [https://docs.chain.link/cre/organization/linking-keys](https://docs.chain.link/cre/organization/linking-keys) — This page explains the process of linking blockchain wallet addresses to a Chainlink CRE organization, allowing users to deploy and manage workflows. It covers automatic and manual linking methods, prerequisites such as having a funded wallet and the necessary CLI setup, and constraints like the maximum number of keys per organization. Important commands include 'cre account link-key' for linking and 'cre account unlink-key' for unlinking keys, with caveats about the destructive nature of unlinking.. Key details: One organization per address: Each wallet can only be linked to one CRE organization.; Maximum of 2 keys per organization.; Command to link a key: cre account link-key --target <target-name>; Command to unlink a key: cre account unlink-key --target <target-name>; Private key must be set in .env file as CRE_ETH_PRIVATE_KEY.; Linked addresses are visible via cre account list-key.
