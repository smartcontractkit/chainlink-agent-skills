# Getting Started

- [https://docs.chain.link/data-feeds/getting-started](https://docs.chain.link/data-feeds/getting-started) — Tutorial for consuming a Chainlink Price Feed from a Solidity contract on Sepolia testnet. Key details: Solidity pragma ^0.8.7; import `@chainlink/contracts/src/v0.8/shared/interfaces/AggregatorV3Interface.sol`; constructor sets `dataFeed = AggregatorV3Interface(<proxy address>)`; call `dataFeed.latestRoundData()` returning `(roundId, answer, startedAt, updatedAt, answeredInRound)`; example uses Sepolia BTC/USD proxy `0x1b44F3514812d835EB1BDB0acB33d3fA3351Ee43` with 8 decimals. Deploy via Remix with Injected Provider (not JavaScript VM — contract calls another onchain contract). Caveats: example is unaudited/not for production; on L2s (Arbitrum/OP/Metis) check the L2 Sequencer Uptime Feed; decimals are feed-specific.
